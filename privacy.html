나의 레슨일지 (My Lesson Diary)

최종 수정일: 2026년 3월 10일

1. 기본 원칙

나의 레슨일지는 사용자의 데이터를 앱 운영자의 서버에 저장하지 않습니다.
모든 데이터는 사용자 본인의 기기(브라우저) 또는 사용자 본인의 Google 계정에만 저장됩니다.

본 방침은 나의 레슨일지(이하 "앱")가 수집하는 정보와 그 사용 방식을 설명합니다.

2. 수집하는 정보

앱은 다음 정보를 처리합니다:

  • 학생 정보 — 이름, 학교, 학년, 연락처, 전공, 수업 일정 등 사용자가 직접 입력한 정보
  • 수업 기록 — 수업 날짜, 내용, 숙제, 수업료 납부 여부 등 사용자가 직접 입력한 정보
  • Google 계정 정보 — Google 로그인 시 이름과 이메일 주소 (Google Drive 연동 목적)

앱은 위 정보 외에 어떠한 개인정보도 수집하지 않습니다.

3. 정보의 저장 위치

  • Google 미연동 시 — 모든 데이터는 사용자 기기의 브라우저 로컬 저장소(localStorage)에만 저장됩니다. 앱 운영자는 이 데이터에 접근할 수 없습니다.
  • Google Drive 연동 시 — 데이터는 사용자 본인의 Google Drive 앱 전용 폴더(appDataFolder)에 저장됩니다. 앱 운영자는 이 데이터에 접근할 수 없습니다.
  • Google Calendar 연동 시 — 수업 기록이 사용자 본인의 Google 캘린더에 이벤트로 등록됩니다. 앱 운영자는 이 데이터에 접근할 수 없습니다.

4. Google API 사용

앱은 다음 Google API를 사용합니다:

Google Drive API Google Calendar API Google OAuth 2.0

Google 로그인은 선택 사항이며, 로그인하지 않아도 앱의 모든 기능을 사용할 수 있습니다.

Google API를 통해 수집된 정보는 앱의 핵심 기능(데이터 동기화, 캘린더 등록) 제공 목적으로만 사용되며, 제3자와 공유하거나 광고 목적으로 사용하지 않습니다.

본 앱의 Google API 사용은 Google API 서비스 사용자 데이터 정책을 준수합니다.

5. 제3자 공유

앱은 사용자의 데이터를 어떠한 제3자에게도 판매, 공유, 또는 제공하지 않습니다.

6. 데이터 보관 및 삭제

  • 로컬 데이터는 사용자가 브라우저 데이터를 초기화하면 삭제됩니다.
  • Google Drive 데이터는 사용자가 직접 Google Drive에서 삭제할 수 있습니다.
  • Google Calendar 이벤트는 앱에서 수업 기록 삭제 시 함께 삭제되거나, 사용자가 Google 캘린더에서 직접 삭제할 수 있습니다.

7. 쿠키 및 추적

앱은 쿠키를 사용하지 않으며, 사용자 행동을 추적하거나 분석하는 도구를 사용하지 않습니다.

8. 방침 변경

개인정보처리방침이 변경될 경우 이 페이지에 업데이트됩니다. 중요한 변경이 있을 경우 앱 내 공지를 통해 안내합니다.
